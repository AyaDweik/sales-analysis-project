Sales Analysis Project (Python – Pandas, Matplotlib, Seaborn)
#Project Overview

This project analyzes sales data to extract insights about:

Customer spending behavior

Revenue per product category

Total revenue per order

Quantity distribution

Most valuable customers

It demonstrates data cleaning, transformation, visualization, and exporting.

# Files in This Repository

Sales_Analysis.ipynb – Full project notebook

cleaned_sales.csv – Final cleaned dataset

visualizations PNG charts generated in the analysis

# Skills Demonstrated
- Data Cleaning

Handling missing values

Filling missing numeric values with mean

Replacing missing text labels

- Data Analysis

Creating new metrics (total revenue)

Grouping & aggregation

Category-level revenue

- Data Visualization

Bar charts (Seaborn)

Histograms

Business insights interpretation
